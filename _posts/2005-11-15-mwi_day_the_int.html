The Integrated Operator's Perspective on Mobile Web, Keith Waters of France Telecom

FT does mobile, web and internet, and recognises the value of cross-platform delivery.
One web is an ultimate goal, not a solved problem
Profileration of devices: author once, publish many
Increased mobility: right content in the right place at the right time

Mobile web of tomorrow != desktop PC of today

Two examples of consistent user experiences:
- web mini-applications (e.g. desktop widgets)
- consistent user experience from desktop to mobile
- apple widgets can be used on mobile

- browser based application to display LBS info and signal strength
