--- layout: post title: Amnesty releases API access to censored files date: 2006-10-27 00:00:00 category: digital-media ---

Amnesty releases API access to censored files: "The irrepressible.info campaign, launched in May, provides javascript banners featuring censored sites and content. The API will give direct access to the irrepressible.info site’s database of censored material so that supporters can create their own applications to distribute the information better."
